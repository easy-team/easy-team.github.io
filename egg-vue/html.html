# HTML前端渲染

# 背景


在 前端渲染模式 和 asset 渲染模式 章节讲到了基于 React 的前端渲染模式，但都依赖  egg-view-react-ssr 插件，那如何基于已有 egg 模板引擎 (egg-view-nunjucks 或 egg-view-ejs) + Webpack 完全自定义前端方案呢？


# 关键问题


  • 本地开发 Webpack 与 Egg 集成，可以直接使用 egg-webpack， 该插件可以独立使用, 当然你可以实现。
  • 通过 egg-view-nunjucks 或 egg-view-ejs 插件进行数据绑定
  • 通过 html-webpack-plugin  插件生成 HTML 文件，并自动注入 JS/CSS 依赖
  • 本地开发时，需要通过 write-file-webpack-plugin  插件把 Webpack HTML 文件写到本地。Webpack 默认是在内存里面，无法直接读取。
