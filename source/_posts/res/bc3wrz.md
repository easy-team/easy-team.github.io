## 构建文件


#### TypeScript Node 构建

```bash
npm run tsc
```



#### TypeScript 前端工程构建

```bash
npm run build
```


#### 项目打包部署

1. 先运行 `npm run tsc` 和 `npm run build` 构建 TypeScript Node 代码和 TypeScript 前端代码
1. 项目代码和构建代码一起打包代码
1. 应用部署后，通过 `npm start` 启动应用
